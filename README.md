# 👨‍💻 Eliseu Francisco de Souza

- 🧩 Criação de sistemas completos com **autenticação JWT**, integração com **banco de dados MySQL** e implantação em **AWS**
- 💻 Atuação como **Suporte N1**, desenvolvimento web e manutenção de sistemas
- 🎨 Experiência com **design gráfico** e **marketing digital**, inclusive com gestão de **marketplaces (Mercado Livre, Shopee)**
